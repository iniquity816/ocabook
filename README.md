This repository contains the files for *A World of Cellular Automata - Beneath the Surface*, almost as they were on March 26th, 2026. Some minor changes have been made (some of the images no longer exist). The book intends to serve as an unofficial counterpart, or sequel to, *Conway's Game of Life - Mathematics and Construction* by Nathaniel Johnston and Dave Greene, focusing instead on the larger variety of objects present in other cellular automaton rules.
